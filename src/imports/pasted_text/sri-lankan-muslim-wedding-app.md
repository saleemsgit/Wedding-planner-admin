# Premium Sri Lankan Muslim Wedding Planning & Booking Platform – UI/UX Design Prompt

Design a modern, luxurious, mobile-first web application UI for a Sri Lankan Muslim wedding planning and booking platform.

The platform should help users plan, organize, and book every wedding-related requirement from engagement to Walima while reflecting authentic Sri Lankan Muslim wedding culture and traditions.

---

# CORE CONCEPT

The platform should function as a complete A–Z wedding planning ecosystem where users can:

* Discover wedding vendors
* Book wedding services
* Plan events step-by-step
* Track wedding progress
* Manage budgets
* Organize cultural traditions
* Coordinate family ceremonies
* Manage gift exchange events
* Track wedding tasks and timelines

The UI should feel:

* Elegant
* Premium
* Cultural
* Emotional
* Clean and modern
* Minimal but luxurious
* Inspired by high-end wedding brands and modern SaaS dashboards

---

# CULTURAL REQUIREMENTS (VERY IMPORTANT)

The design must strongly reflect Sri Lankan Muslim wedding culture.

Include dedicated planning sections for:

1. Proposal / Engagement
2. Seer / Jahez Gift Exchange
3. Mehndi Night
4. Groom Arrival
5. Nikah Ceremony
6. Wedding Reception
7. Walima

The platform should support both bride-side and groom-side wedding planning.

Include cultural features such as:

* Gift tray planning

* Traditional food planning
* Gold jewelry planning
* Wedding card ordering
* Mehndi event planning

* Wedding package customization
